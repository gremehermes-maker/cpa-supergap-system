# 1.2 1차 문제집 객관식 슬라이싱 (1.2_mcq_slicer.py)

## 1. 개요
객관식 문제집을 '문항' 단위로 쪼개는(Slicing) 교재 처리 모듈입니다. 이미지 슬라이싱 시스템을 통해 개별 문항 번호와 해설이 각각 독립된 노드로 추출되도록 지원합니다.

## 2. 상세 작업 (Phase 0 크롭 파이프라인)
- **정밀 크롭:** 객관식 문제집의 문제 영역과 해설 영역을 정밀하게 크롭하여 나눕니다.
- **독립 노드화:** 문제 영역과 해설 영역이 마진노트 내부에서 1:1로 매칭되거나 독립적인 노드로 임포트될 수 있도록 구조화합니다.

## 3. 제약사항 및 한계
- 텍스트 OCR에 전적으로 의존하지 않으며, 이미지를 잘라내는(Slicing) 형태에 초점을 맞추어 오류(Data Poisoning)를 원천 방어합니다.
