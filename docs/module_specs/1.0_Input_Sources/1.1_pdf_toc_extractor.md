# 1.1 기본서 PDF 북마크 추출 (1.1_pdf_toc_extractor.py)

## 1. 개요 및 설계 철학
개념서 역할을 하는 기본서 PDF 파일에서 오직 '목차(TOC)'를 추출하여 시스템 전체의 뼈대(OPML)를 만드는 모듈입니다.

과거처럼 PDF 본문 텍스트를 강제로 긁어와 무분별하게 자동 Anki 카드로 만들던 오류(Card Bloat) 유발 로직은 완전히 폐기했습니다. 대신 이 모듈은 오직 교재의 '목차(TOC)'만 추출하여 전체 시스템의 뼈대(OPML)를 잡는 구조화 용도로만 사용됩니다.

## 2. 주요 기능 및 파이프라인 (Phase 0)
- **PDF 파싱:** PDF 문서의 내장 북마크(TOC) 객체를 파이썬 스크립트로 추출하여 JSON 형태로 파싱합니다.
- **OPML 생성:** 추출된 목차 데이터를 바탕으로 MarginNote 4 및 Logseq에서 Import 할 수 있는 OPML 뼈대 파일을 생성합니다.

## 3. 입출력 스펙
- **입력:** 기본서 원본 PDF 파일
- **출력:** 계층화된 목차 구조 JSON 및 OPML
- **연계:** 추출된 목차는 `1.4_toc_url_mapper.py` 로 넘어가 마진노트 고유 URL과 매핑됩니다.
