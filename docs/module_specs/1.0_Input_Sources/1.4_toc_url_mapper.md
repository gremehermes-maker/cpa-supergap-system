# 1.4 TOC-URL 매핑 DB (1.4_toc_url_mapper.py)

## 1. 개요 및 설계 철학 (시스템 근본 해답 1)
기본서의 목차(예: `1.1.1 자산손상`)와 MarginNote URL(`marginnote4app://`)을 1:1로 묶는 단일 진실 공급원입니다. 사용자가 수동으로 URL을 복사해 붙여넣는 '스파게티 선긋기 노동'을 해방시킵니다.

## 2. 개정판 출시에 따른 연결망 붕괴 방어 (Mitigation 9.2.3)
- **한계:** 오답 연결 시 목차 번호에 과도하게 의존하면, 내년 교재 개정 시 번호가 밀려 기존 Obsidian 약점 랭킹 통계와 연결망이 붕괴될 위험이 존재했습니다(Stale Data).
- **방어 로직:** 파이썬 스크립트나 프롬프트 내부에 `1.1.1` 같은 번호를 하드코딩하지 않습니다. 모든 매핑은 `TOC_Mapping.json` 파일에 저장됩니다. 교재가 개정되면 에이전트나 사용자가 이 JSON 내의 번호표만 업데이트하면, 수만 개의 기존 연결망 훼손 없이 즉각 동기화됩니다.
