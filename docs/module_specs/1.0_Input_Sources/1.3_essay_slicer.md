# 1.3 2차 연습서 주관식 슬라이싱 (1.3_essay_slicer.py)

## 1. 개요
가장 복잡한 구조를 가진 2차 주관식 연습서 교재를 처리하는 모듈입니다. 하나의 긴 시나리오(공통 자료) 밑에 여러 개의 '새끼 물음(물음 1, 물음 2...)'이 달리는 계층형 구조(Hierarchy)를 분할합니다.

## 2. 위계 파싱 아키텍처 (계층형 Slicing)
- **공통자료와 물음의 분할:** [공통 자료]를 부모 노드로 두고, [물음 1], [물음 2], [해설]을 자식 노드로 묶어 슬라이싱합니다.
- **과거 한계점(Logseq 탭 의존성)의 극복:** 기존에는 Logseq에서 사용자가 탭(Tab) 키로 들여쓰기하여 위계를 잡았으나(9.5.1 이슈), 현재는 마진노트의 시각적 묶음(`ZGROUPNOTEID`)을 진실의 공급원으로 사용하므로, 1.3 모듈은 시각적 이미지를 계층적으로 임포트할 수 있도록 데이터를 정제하는 데 집중합니다.
