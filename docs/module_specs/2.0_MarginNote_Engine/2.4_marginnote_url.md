# 2.4 marginnote4app 고유 URL (2.4_marginnote_url.py)

## 1. 개요
모든 마진노트 노드가 갖는 영구적인 딥링크(`marginnote4app://...`)의 추출 및 연결 사양서입니다. 외부 앱 어디서든 터치 시 원문 해당 위치로 즉시 워프(Warp)합니다.

## 2. 데드 링크 폴백 아키텍처 (Fallback URL - Mitigation 7.3)
- **문제점:** 기기간 마크다운 파일 동기화가 깨져서 옵시디언 로컬 딥링크(`obsidian://`)가 끊어지는 현상.
- **방어 로직:** 모든 Anki 카드 뒷면 하단 및 Logseq 노드 메타데이터에 이 `marginnote4app://` 고유 URL을 강제로 박아 넣습니다. 이를 통해 로컬 파일 링크가 죽더라도 언제든 마진노트 원문으로 살아 돌아갈 수 있는 안전망(Fallback)이 100% 보장됩니다.
- **종속성 리스크 (9.1.1):** 애플 생태계의 스킴 종속성은 변경 불가능한 외부 리스크이나, 가장 효율적이므로 현재는 수용(Accept)된 상태입니다.
