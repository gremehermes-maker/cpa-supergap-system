# 2.1 마인드맵 & 노드 (2.1_mindmap_node.py)

## 1. 개요
마진노트 4의 핵심인 마인드맵 블록(노드) 구조화 및 SQLite DB 통신 사양서입니다. 교재 내용과 사용자의 필기, 그리고 오답 연결이 모이는 원천 창고 역할을 합니다.

## 2. 마진노트 SQLite DB (`ZBOOKNOTE` 테이블) 스펙
스크립트가 직접 읽어들이는 `.marginnotes` 파일 내 주요 컬럼입니다. (라이브 DB 충돌 방지를 위해 파이썬은 `/tmp/` 폴더로 스냅샷 복사된 DB를 읽어옵니다.)

- `ZNOTEID`: 노드 고유 ID (UUID). Anki 카드 중복 방지(멱등성)에 사용됨.
- `ZNOTETITLE`: 노드 제목
- `ZHIGHLIGHT_TEXT`: PDF 발췌 텍스트
- `ZGROUPNOTEID`: 부모 노드 ID (계층 구조)
- `ZMINDLINKS`: 자식 노드들의 ID 리스트 (`|`로 구분). '부모-자식' 관계 추적에 사용.
- `ZNOTES`: 사용자 메모 (blob)
