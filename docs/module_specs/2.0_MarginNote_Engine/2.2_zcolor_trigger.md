# 2.2 ZCOLOR 색상 트리거 (2.2_zcolor_trigger.py)

## 1. 개요 및 설계 철학 (초격차 단순화)
텍스트 파싱 오류와 타이핑 스트레스를 완벽히 해결한 메타인지 기록 기술입니다. 사용자가 키보드 타이핑(예: `주의:`, `절차:`) 없이 오직 펜슬 색상 변경만으로 시스템과 상호작용합니다. (문서 9.5.3 참조)

## 2. SQLite 트리거 매핑 스펙
마진노트 `ZBOOKNOTE` 테이블 내의 `ZCOLOR` 인덱스 숫자를 기반으로 텍스트 프리(Text-Free) 자동화를 구현합니다.
- **Index 2 (빨간색):** 무조건 `⚠주의(함정)` 카드로 생성
- **Index 3 (초록색):** 무조건 `📋절차` 카드로 생성

## 3. 레거시 트리거 (ZBOOKCOMMENT 파싱)
색상 도입 이전, `ZBOOKCOMMENT` 테이블의 `ZCOMMENTTEXT` 첫 줄을 읽어 파싱하던 9대 특수 트리거 규칙(강사, 공식, 분개, 빈출 등)도 레거시 지원을 위해 스크립트 내에 유지됩니다.
