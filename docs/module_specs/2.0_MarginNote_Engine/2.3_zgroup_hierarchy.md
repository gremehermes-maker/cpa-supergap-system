# 2.3 ZGROUPNOTEID 위계 (2.3_zgroup_hierarchy.py)

## 1. 개요
마진노트 내 사용자의 네모 박스 묶음 행위(Grouping)를 바탕으로 데이터의 계층 구조를 도출하는 모듈입니다.

## 2. 탭(Tab) 파싱 의존성의 완전한 제거 (Mitigation 9.5.1)
- **과거 한계:** Logseq에서 사용자가 실수로 탭을 잘못 치면 2차 연습서의 부모(공통자료)-자식(물음) 구조가 와해되었습니다.
- **방어 로직:** 파이썬 스크립트는 Logseq의 텍스트가 아닌, 마진노트의 `ZGROUPNOTEID`(부모 노드)와 `ZMINDLINKS`(자식 노드 리스트)를 '단일 진실 공급원'으로 취급합니다. 시각적으로 묶인 마인드맵 구조를 그대로 읽어와 Logseq용 마크다운으로 역변환(자동 들여쓰기) 해줍니다.
