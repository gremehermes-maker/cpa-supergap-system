# 7.2 Tutor 봇 : Claude 3.5 (7.2_tutor_bot.py)

## 1. 개요
회계학 및 재무관리 전문 지식 질문에 답변하는 주력 튜터 에이전트 (CPU 1) 입니다.

## 2. 전문 용어 STT 변환의 품질 저하 방어 (Mitigation 9.3.1)
- **위험성:** 회계 강의 녹음(STT) 과정에서 강사의 억양 등으로 발생한 오탈자("자본유지 젓갈법")가 RAG DB에 적재되어 환각을 유발하는 문제 (Data Poisoning).
- **해결책 (Ground-Truth 기반 오토 코렉트):** 교재 PDF 텍스트에서 '사용자 전용 용어 사전'을 자동 추출합니다. STT 텍스트가 ChromaDB(RAG DB)에 적재되기 전, 소형 LLM이 이 사전을 기반으로 오탈자를 강제 치환 및 정제합니다.

## 3. 통신 규약
- 단독으로 사용자에게 답변하지 않으며, 체커 봇(7.3)의 승인을 거쳐야만 최종 답변이 노출됩니다.
