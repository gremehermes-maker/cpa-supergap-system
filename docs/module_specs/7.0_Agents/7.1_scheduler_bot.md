# 7.1 스케줄러 & Inbox 봇 (7.1_scheduler_bot.py)

## 1. 개요
메신저 봇 기능과 더불어 LLM OS의 Memory Controller 역할을 수행하는 핵심 비동기 통신 에이전트입니다.

## 2. Memory Editor (RAM -> Disk 아키텍처)
- **배경 (Phase 5):** 1만 토큰의 긴 대화를 매번 읽히지 않기 위해 역할을 분리했습니다.
- **미세 컨텍스트 소실 방어 (Mitigation 9.3.2):** Memory Editor는 세션 종료 시 슬랙 대화 로그를 모니터링하다 핵심 정수만 뽑아 하드디스크(`LLM_Wiki/User_State.md`)에 저장합니다. 이때 산문형 한 줄 요약을 금지하고, 반드시 아래와 같이 구조화된 포맷 출력을 강제합니다.
  - `[약점 개념]`: 단기매매금융자산
  - `[사용자의 실수 원인]`: 평가손익을 당기손익이 아닌 OCI로 잘못 인식함

## 3. 스케줄 조정
- 일상 캘린더 모니터링 후 Anki 듀데이트를 물리적으로 승인/조정합니다.
