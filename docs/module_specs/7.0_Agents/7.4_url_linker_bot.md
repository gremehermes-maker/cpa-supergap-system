# 7.4 URL 매핑 연결 봇 (7.4_url_linker_bot.py)

## 1. 개요
사용자가 틀린 문제와 개념서의 원문 위치를 조용히 이어주는 백그라운드 스파게티 연결 해결 봇입니다. (마진노트 프록시 노드 생성 방식을 완벽히 대체)

## 2. 인지적 과부하 방지 UX (Mitigation 9.5.2)
- **과거의 문제:** 오답을 연결할 때 목차 번호(예: `1.1.1`)를 외우고 타이핑해야 하는 스트레스.
- **해결책 (자동 추천 UX 도입):** 에이전트가 연습서 문제의 텍스트("상각원가")를 추출하여 교재 목차 DB와 유사도 벡터 검색을 수행합니다. 
- **작동 방식:** 봇이 슬랙에 **"🤖 힌트: 이 문제는 [6.3.2 당기손익-공정가치측정] 개념과 유사합니다. 연결할까요?"** 라며 상위 3개의 후보 버튼과 `[🔍 직접 찾기]` 버튼을 띄워 사용자 편의성을 극대화합니다.
