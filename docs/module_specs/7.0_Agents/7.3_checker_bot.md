# 7.3 Checker 봇 : Qwen / Gemini 1.5 (7.3_checker_bot.py)

## 1. 개요
Tutor 봇의 환각(Hallucination)을 막기 위한 크로스 체크 에이전트 (환각 방지 필터) 입니다.

## 2. 이기종 모델 교차 검증 아키텍처
- **목적:** 논리력이 뛰어난 Claude(Tutor)와 지식 베이스 팩트체크가 강한 Qwen/Gemini(Checker)가 서로 상호 감시하여 환각을 원천 차단합니다.
- **규칙:** 체커 봇은 검증 시 반드시 **IFRS 기준서 등 문단 번호를 색인**하여 답변의 근거를 제시해야 합니다.

## 3. 무한 루프 교착 상태 (Mitigation 9.3.3)
- Claude와 Qwen이 서로 합의하지 못하고 교착 상태에 빠지는 것을 막기 위한 서킷 브레이커 로직은 현 단계에서 사용자 피로도를 고려하여 보류(Pass)된 상태이므로, 에러 발생 시 타임아웃 처리가 필요합니다.
