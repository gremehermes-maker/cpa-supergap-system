# 4.1 Logseq 2차 문제 아웃라이너 (4.1_logseq_outliner.py)

## 1. 개요
'숲의 나무'를 관리하는 공간으로, 2차 주관식 연습서 등 복잡한 계층 구조를 담아내는 아웃라이너 마크다운 연동 모듈입니다.

## 2. 계층 통제 (다중 위계 파싱)
- **기존 문제점:** 2차 연습서의 날것 구조(`공통자료 → 물음1 → 해설`)가 그대로 Anki로 전송되면 맥락을 완전히 상실합니다.
- **방어 설계 (Mitigation 9.5.1):** 
  사용자가 Logseq에서 직접 들여쓰기(Tab)를 치는 노동을 없앴습니다. 마진노트의 본원적 계층 구조(`ZGROUPNOTEID`)를 최우선 진실로 삼아 읽어들인 뒤, 파이썬 스크립트가 알아서 탭(들여쓰기) 된 형태의 마크다운 포맷으로 변환 저장합니다.
- **역할:** [공통 자료]를 부모로, [물음 1]을 자식으로 두는 직관적 아웃라이너 구조로 변환되어 연습서의 위계를 보존합니다.
