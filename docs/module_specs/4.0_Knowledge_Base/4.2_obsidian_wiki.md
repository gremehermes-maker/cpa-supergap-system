# 4.2 Obsidian LLM Wiki (4.2_obsidian_wiki.py)

## 1. 개요 (메타인지 관제실)
숲 전체를 조망하고, 약점 통계를 내며, 향후 LLM 에이전트의 지식 베이스(하드디스크) 역할을 하는 관제탑 모듈입니다.

## 2. 궁극의 오답 통계 (진실의 방 대시보드)
- AnkiDroid에서 사용자가 누른 오답 횟수(Lapses)를 취합하고, 헤르메스 봇이 만들어준 '약점 연결망(마크다운 프론트매터)'을 기반으로 Dataview 플러그인과 Graph View를 구동시킵니다.
- 이를 통해 "나의 2차 연습서 약점 랭킹 (예: 1위 자산손상 27번 틀림)"을 실시간 대시보드로 시각화하여 명확한 메타인지를 제공합니다.

## 3. 영구적인 지식 축적 (LLM OS 아키텍처)
- **Persistent Memory (Disk):** 안드레 카파시의 LLM OS 개념에 따라, Memory Editor 에이전트가 세션의 핵심 요약 내용(사용자의 약점, 학습 습관 등)을 이곳 Obsidian의 `LLM_Wiki/User_State.md`에 구조화된 JSON/YAML 형태로 '던져넣어' 데이터를 영구 보존합니다.
