# TR_3.3_to_5.1_anki_push.py

## 목적 및 역할
추출된 문제를 Anki-Connect API를 통해 Anki 스케줄러 환경으로 전송하여 카드를 생성합니다.

---
## 💡 TR 브릿지 개발자 공통 규약
1. **단일 책임 원칙:** 이 브릿지는 출발지 번호의 아웃풋을 목적지 번호의 인풋 포맷으로 1:1 변환하는 일만 수행해야 합니다.
2. **에러 핸들링:** 변환 과정에서 데이터 누락이나 포맷 파싱 에러 발생 시, 침묵 실패(Silent fail)하지 않도록 반드시 Slack/Telegram 봇(6.2 또는 7.1)으로 예외를 던져 알람을 송출하십시오.
