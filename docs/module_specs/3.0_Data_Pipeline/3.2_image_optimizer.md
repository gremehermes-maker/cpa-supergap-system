# 3.2 이미지 최적화 / Webp (3.2_image_optimizer.py)

## 1. 개요
Anki 데이터 동기화 폭발을 막기 위한 강제 해상도 축소 및 압축 모듈입니다.

## 2. 상세 스펙 (Phase 1~2 마스터플랜)
- **목적:** 스마트폰(모바일) 환경에서 수천 장의 Anki 이미지 카드로 인해 용량 초과와 로딩 지연이 발생하는 것을 해결합니다.
- **주요 로직 (WebP 강제 변환):** 교재 스크린샷, 문제 캡처 등을 Anki 미디어 폴더로 복사하기 전, 파이썬 스크립트가 해상도를 모바일 친화적으로 강제 축소하고 `.webp` 포맷으로 압축합니다. 이를 통해 동기화 지연과 용량 폭발을 80% 이상 절감합니다.
