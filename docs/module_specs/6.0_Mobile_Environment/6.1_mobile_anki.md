# 6.1 모바일 Anki 뷰어 (6.1_mobile_anki.py)

## 1. 개요
스마트폰(Galaxy AnkiDroid) 환경에서의 외출 및 이동 중 학습을 전담하는 모바일 환경 규약입니다.

## 2. 단방향 덮어쓰기 방어 및 절대 원칙 (Mitigation 9.2.1)
- **발생 가능한 문제:** 스마트폰에서 오타를 발견하고 직접 카드 내용을 수정한 뒤 PC와 동기화하면, 충돌 병합 없이 PC의 원본 데이터로 덮어씌워져 모바일 수정 내역이 완전히 날아갑니다 (데이터 증발).
- **시스템 절대 원칙:** 안드로이드 폰에서는 오직 읽고, 복습(맞음/틀림 버튼 터치)만 수행합니다. **절대 폰 내에서 카드의 내용을 직접 수정하지 않습니다.** 수정 소요가 발생할 경우 6.2 텔레그램 Inbox 패턴을 사용합니다.
