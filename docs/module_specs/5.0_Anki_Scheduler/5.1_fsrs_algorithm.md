# 5.1 FSRS 스케줄러 알고리즘 (5.1_fsrs_algorithm.py)

## 1. 개요
전통적 SM-2를 넘어선 최신 AI 망각 곡선인 FSRS 스케줄러 알고리즘을 연동하고 외부 간섭으로부터 보호하는 모듈입니다.

## 2. 알고리즘 훼손 방어 (Mitigation 9.4.2)
- **과거 한계:** 7.1 스케줄러 에이전트가 SQLite 상의 `due`(복습 예정일)를 강제로 밀어버리면 FSRS 알고리즘이 붕괴하여 다음 주기가 기하급수적으로 뻥튀기되는 현상.
- **방어 설계:** 에이전트나 파이썬 스크립트는 절대 DB 원시 데이터(`due`)를 직접 조작할 권한이 없습니다. 오직 Anki 공식 생태계의 **'FSRS Helper 연기(Postpone) 기능' API**만 호출하도록 허용되어 스케줄의 과학적 무결성을 지킵니다.

## 3. 무분별한 카드 증식 방어 (Mitigation 9.4.1)
- 정의(`~은 ~이다`) 자동 감지에 의한 빈칸 카드 자동 생성 로직은 삭제(폐기)되었습니다.
- `절차:` 트리거 사용 시 과거처럼 카드를 4장으로 쪼개지 않고 단 1장의 카드만 생성하며, Javascript 템플릿을 통해 화면 터치 시 점진적으로 텍스트가 공개(Progressive Reveal) 되도록 최적화했습니다.
