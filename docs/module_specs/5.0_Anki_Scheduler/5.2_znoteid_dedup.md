# 5.2 ZNOTEID 중복 방지 (5.2_znoteid_dedup.py)

## 1. 개요
파이썬 스크립트의 반복 실행에 따른 Anki 카드의 중복 증식을 막는 방어(멱등성) 모듈입니다.

## 2. 카드 중복 방지 및 멱등성 (Idempotency) 로직
- 모든 Anki 카드의 뒷면 숨긴 필드(Hidden Field)에 마진노트의 고유 `ZNOTEID` (UUID)를 기록합니다.
- 추출 스크립트가 수백 번 반복되어도, 이 필드를 스캔하여 이미 존재하는 노트는 내용만 스마트하게 덮어쓰기(업데이트) 하므로 무한 증식을 방지합니다.

## 3. 수동 조작 금지 원칙 (Mitigation 9.2.4)
- **금지 사항:** 사용자가 Anki Desktop 내에서 수동으로 '카드 복사' 또는 '잘라내기(Split)' 기능으로 카드를 건드릴 경우, 이 UUID가 파괴되거나 복제되어 데이터가 꼬이게 됩니다.
- **원칙:** Anki 내에서는 시스템 생성 카드를 수동으로 분리/복사하는 행위가 절대 금지되며, 분할이 필요하면 오직 마진노트(원본 창고)에서 분할해야 합니다.
